# UEFA Champions League 2023/2024 Teams Web Map


Welcome to the UEFA Champions League 2023/2024 Teams Web Map repository! This project is designed to showcase the locations of the 32 teams participating in the UEFA Champions League for the 2023/2024 season using Leaflet, a popular JavaScript mapping library.

## Features

- Interactive map displaying the locations of all 32 teams participating in the UEFA Champions League.
- Custom markers and pop-up information for each team.
  - Number of titles won by each team.
  - Stadium name and spectators capacity.
  - UEFA ranking.
  - Market value (automatically updated work in progress).
- User-friendly interface with zoom and pan functionality.

## Demo

You can check out a live demo of this web map here: [UEFA Champions League 2023/2024 Teams Web Map Demo](https://webmappergis.github.io/Uefa-Champions-League-Map/).



## Usage

- Click on a team marker to view more information about that team.
